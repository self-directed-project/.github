# 🎀 개요

이 프로젝트는 회의실 예약 웹 서비스입니다. 주요 기능은 다음과 같습니다:
<br />

- **회의실 예약 등록 및 수정**: 패아지 별 모달 기능을 통해 예약 CRUD 기능
- **캘린더를 통한 예약 시각화**: react-calendar를 통한 캘린더 커스텀
- **회의실 즐겨찾기 등록**: 회의실 별 즐겨찾기 등록
- **나의 예약 현황 보기**: 전체 회의 및 나의 예약 현황 시각화
